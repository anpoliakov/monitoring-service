# ДЗ №1
Cервис для подачи показаний счетчиков отопления, горячей и холодной воды

## Реализовано
- подача показаний один раз в месяц
- пользователь может видеть только свои показания, администратор - показания всех пользователей
- предусмотрена возможность расширения перечня подаваемых показаний
- данные хранятся в памяти приложения
- консольное приложение
- регистрация пользователя
- авторизация пользователя
- эндпоинт для получения актуальных показаний счетчиков
- эндпоинт подачи показаний
- эндпоинт просмотра показаний за конкретный месяц
- эндпоинт просмотра истории подачи показаний
- контроль прав пользователя
- аудит действий пользователя (авторизация, завершение работы, подача показаний, получение истории подачи показаний и тд)
